# Blocks

The submodule projects showcased on the Perspective README.md, as well as a
simple `@finos/perspective-viewer-datagrid` custom renderer for hosting the
projects locally.

## Installation / Run

You must clone Perspective with the `--recurse-submodules` flag

```bash
git clone --recurse-submodules https://github.com/finos/perspective.git
```

or 

```bash
git submodule update
```

You can then run the example server from teh standard perspective `start` task:

```bash
yarn start blocks
```

Then navigate to [http://localhost:8080/](http://localhost:8080/)
